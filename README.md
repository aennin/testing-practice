# Testing Practice

This project is a JavaScript testing exercise focused on writing unit tests using **Jest** and practicing **test-driven development (TDD)**.

It is based on common utility functions often used to learn testing fundamentals.

## Technologies Used

- JavaScript (ES6+)
- Jest
- Babel

##  Functions Implemented

### 1. `capitalize`
Capitalizes the first character of a string.

### 2. `reverseString`
Reverses a given string.

### 3. `calculator`
An object that performs basic arithmetic operations:
- add
- subtract
- multiply
- divide

### 4. `caesarCipher`
Applies a Caesar cipher shift to a string while preserving case and punctuation.

### 5. `analyzeArray`
Takes an array of numbers and returns an object containing:
- average
- min
- max
- length

## Running the Tests

1. Install dependencies:
```bash
npm install
```
2. Run tests:
```bash
npm test
```

## Purpose
The goal of this project is to:
- Practice writing unit tests
- Learn Jest fundamentals
- Understand test-driven development
- Improve code quality and reliability
